Frequently Asked Questions

What are Copilots?

A Copilot / AI coding assistant is a tool that uses artificial intelligence to help you write code more efficiently and accurately. It's like having a super-smart code reviewer, autocomplete, and integrated chat tool all in one.

Where are Copilots Available?

Many modern IDEs, such as Visual Studio Code, JetBrains IntelliJ IDEA, and Eclipse, offer plugins or extensions that integrate AI-powered coding assistance features like intelligent code completion, code suggestions, and error detection.

For more general use, Microsoft is making Copilots available in tools such as Word, Excel, PowerPoint and Outlook as well.

What kind of things can users do with Copilots?

Copilot can generate code snippets based on context and user input, offer insights into APIs, libraries, and programming patterns, help identify potential errors or bugs in code, and assist in refactoring code by offering alternative implementations or suggesting optimizations.

Microsoft copilot can be used within browser. Using Code Copilots within other web-based code editors or browsers may not be directly supported.
